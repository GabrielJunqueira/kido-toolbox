How it works

  • Uploads your Deals_Kido.csv or Excel file.
  • Anonymizes Deal and Company names using the salt.
  • Cleans up Country names (removes Notion links).
  • Normalizes text and formats currency values (Amount CHF).
  • Returns a processed CSV file named deals_treated.csv.
